This is a Income Expense Tracker made using Django. The source code of this project can be found on Github.

Users can create expenses based on categories and incomes based on different sources. Users can filter income/expense based on received/spent today, weekly, montly and yearly. All this information is present on dashboard and income/expense summary in form of a Graph. Users can also download this data in form of pdf, csv and excel. Users can also import incomes and expenses from a CSV file. Users also have a Profile where they can change currency to be displayed with expenses and incomes.
